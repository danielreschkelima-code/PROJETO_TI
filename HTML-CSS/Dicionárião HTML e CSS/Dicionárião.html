Não importa como eu escreva entre as tags (onde está simbolizado entre <>), não há importância e o texto sempre seguirá a sua ordem. Isto é, se a tag é de parágrafo, só haverá um paragráfo independentemente de escrever /n.

ALGUMAS TAGS:
<html> = onde inicia tudo
<head> = config do site (UTF,...)
<body> = parte visual
<h1> = cabeçalho principal
<hr> = linha horizontal
<p> = parágrafo
<br> = quebra de linha

*Sempre se encerra o comando com ele mesmo com a adição de uma barra.
*O comando sempre é posto entre <>.
* Tudo o que modifica a disposição dos textos são as tags.
