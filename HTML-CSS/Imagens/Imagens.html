Inserindo Imagens em HTML

É possível inserir imagens em HTML tanto externas (por meio do endereço da imagem) como carragando de um dispositivo local
Observe a imagem de uma bela flor carregada atráves do link de endereço da internet:

*Na tag img, scr é o local da pasta e alt é o texto alternativo (para Pdcs ou imagem não carregada, assecibilidade num geral).
**Como falado anterior mente, é possível colocar uma imagem baixada no dispositivo local. Para tanto, em scr, basta colocar o local dela no disco rígido.
