GUIA DE TERMOS DE PROGRAMAÇÃO

Escrito no dia 6 de junho de 2025..

De acordo com a Wikipédia,

a programação é o processo de escrita, teste e manutenção de um programa de computador. O programa é escrito em uma linguagem de programação, embora seja possível, com alguma dificuldade, o escrever diretamente em linguagem de máquina.

E, quanto tarefa intelectual, contém tantos conceitos quanto se pode imaginar. Dado a complexidade, cabe, por tanto, a este arquivo a explicação simples de alguns desses em prol do fixamento de tal contéudo.

String:
Um(a) vetor/lista de caracteres;
Um tipo de dado capaz de representar texto. Por exemplo, em python: frase = "A janela de vidro craquelou-se."
Laços de repetição:
Uma estrutura que permite a repetições de ações enquanto algo for verdade;
É diferente da estruta condicional.
Programação OO:
Um tipo de programação facil de ser reusado.
Utiliza classes, métodos,...
